Binomial glm model

tidypredict supports glm models with a binomial family and logit link

model <- glm(am ~ mpg + wt, data = mtcars, family = "binomial")
library(tidypredict)




mtcars %>%


  head(10) %>%


  predict_to_column(model) %>%


  select(mpg, wt, am, fit)

How it works

This is that the Tidy Eval formula will looks like for this model

predict_fit(model)
## exp((((mpg) * (-0.324163889357643)) + ((wt) * (-6.41617207291539))) + 
##     (25.8865539927241))/(exp((((mpg) * (-0.324163889357643)) + 
##     ((wt) * (-6.41617207291539))) + (25.8865539927241)) + 1)

Here an example of how the formula will be translated into SQL

fit_to_sql(model, dbplyr::simulate_dbi())
## <SQL> EXP(((("mpg") * (-0.324163889357643)) + (("wt") * (-6.41617207291539))) + (25.8865539927241)) / (EXP(((("mpg") * (-0.324163889357643)) + (("wt") * (-6.41617207291539))) + (25.8865539927241)) + 1.0)
