parse_model

  • Arguments
  • Examples

It parses a fitted R model's structure to extract all of the necessary components to pass along to the functions that create the actual Tidy Eval formula that dplyr can read. It is currently able to parse lm() and glm() models.

parse_model(model)

Arguments

model

An R model object

Examples


df <- data.frame(x = c(1, 2, 5, 6 ,6), y = c(2, 3, 6, 5, 4))
model <- lm(x ~ y, df)
parse_model(model)
